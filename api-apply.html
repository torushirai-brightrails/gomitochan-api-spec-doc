1. 利用条件
- API は「ごみ当ちゃん」が提供するサービスの一部として利用できます。

- API の仕様やレスポンスは予告なく変更される場合があります。
- 必要に応じて、API の提供を通知なしに中止する場合があります。

3. 免責事項

- API 利用者は、オープンデータの利用条件に従ってデータを使用してください。
